## О приложении «Физическая лаборатория»

«Физическая лаборатория» — это веб-приложение для численного решения и визуализации физических задач. Приложение предназначено для образовательных целей и позволяет студентам, преподавателям и исследователям моделировать физические процессы в реальном времени без необходимости написания программного кода.

**Основные возможности:**

- Решение трёх классических физических задач: баллистика с сопротивлением воздуха, стационарная теплопроводность, движение парусной лодки под действием ветра и течения
- Интерактивный ввод параметров через веб-формы
- Визуализация результатов в виде графиков (matplotlib)
- Мгновенный расчёт при изменении входных данных
- Простой и интуитивно понятный интерфейс

**Технологический стек:**

- Backend: Python + фреймворк Bottle
- Frontend: TPL, CSS
- Контроль версий: Git

**Целевая аудитория:**

- Студенты физических и инженерных специальностей
- Преподаватели физики для демонстрации явлений
- Все интересующиеся вычислительной физикой

**Ключевые преимущества:**

- Доступность через браузер без установки дополнительного ПО
- Быстрая итерация расчётов при изменении параметров
- Наглядная визуализация сложных физических процессов
- Открытый исходный код для изучения и модификации

## Установка и запуск

### Требования к окружению

- Python 3.8 или выше
- Фраемворк bottle и библиотека numpy
- pip (менеджер пакетов Python)
- Git (для клонирования репозитория)
- Браузер (Chrome, Firefox, Edge — последние версии)

### Необходимые библиотеки

```bash
pip install bottle numpy
```

### Инструкция по установке и запуску

1. Клонирование репозитория
   
```bash
git clone https://github.com/sqtwix/BottleWebProject_C324_2_PSSH.git
cd BottleWebProject_C324_2_PSS
```

2. Запуск приложения

```bash
python app.py
```

3. Открытие в браузере

Перейдите по адресу: http://localhost:5555
