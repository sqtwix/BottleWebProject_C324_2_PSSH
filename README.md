## О приложении «Физическая лаборатория»

«Физическая лаборатория» — это веб-приложение для численного решения и визуализации физических задач. Приложение предназначено для образовательных целей и позволяет студентам, преподавателям и исследователям моделировать физические процессы в реальном времени без необходимости написания программного кода.

**Основные возможности:**

- Решение трёх классических физических задач: баллистика с сопротивлением воздуха, стационарная теплопроводность, движение парусной лодки под действием ветра и течения
- Интерактивный ввод параметров через веб-формы
- Визуализация результатов в виде графиков (matplotlib)
- Мгновенный расчёт при изменении входных данных
- Простой и интуитивно понятный интерфейс

**Технологический стек:**

- Backend: Python + фреймворк Bottle
- Frontend: TPL, CSS
- Контроль версий: Git

**Целевая аудитория:**

- Студенты физических и инженерных специальностей
- Преподаватели физики для демонстрации явлений
- Все интересующиеся вычислительной физикой

**Ключевые преимущества:**

- Доступность через браузер без установки дополнительного ПО
- Быстрая итерация расчётов при изменении параметров
- Наглядная визуализация сложных физических процессов
- Открытый исходный код для изучения и модификации

## Описание функциональности

Веб-приложение реализует численное решение трёх физических задач:

1. **Движение тела с сопротивлением воздуха**  
   Расчёт траектории, дальности полёта, максимальной высоты и времени полёта с учётом силы сопротивления, пропорциональной скорости.

2. **Стационарная теплопроводность стержня**  
   Решение одномерного уравнения теплопроводности методом конечных разностей с заданными граничными условиями (температура на концах стержня).

3. **Оптимизация движения парусной лодки с учётом ветра и течения**  
   Поиск оптимального угла паруса, обеспечивающего максимальную скорость лодки, с учётом скорости ветра, направления ветра и скорости течения.

Каждая задача представлена отдельной страницей с HTML-формой для ввода параметров, визуализацией результатов в виде графиков (matplotlib) и выводом численных значений.

## Функциональные требования

### Страница «Движение тела с сопротивлением воздуха» (projectile.tpl)

**Поля ввода:**
- начальная скорость v0 (м/с)
- угол к горизонту (град)
- масса тела (кг)
- коэффициент сопротивления воздуха k (кг/м)
- шаг по времени (с)

**Результаты:**
- график траектории y(x)
- дальность полёта (м)
- максимальная высота (м)

**Алгоритм решения:**
- Входные параметры: m (кг), k (кг/м), v0 (м/с), угол (рад), dt (с), g = 9.8 м/с²
- Инициализация: x=0, y=0, vx=v0*cos(угол), vy=v0*sin(угол)
- Циклический расчёт пока y >= 0:
  - Модуль скорости: v = sqrt(vx^2 + vy^2)
  - Ускорения: ax = -(k/m)*vx*v, ay = -g - (k/m)*vy*v
  - Обновление скорости: vx += ax*dt, vy += ay*dt
  - Обновление координат: x += vx*dt, y += vy*dt
- Максимальная высота: max(y)
- Дальность полёта: интерполяция между последней точкой с y>0 и следующей с y<0

### Страница «Стационарная теплопроводность стержня»

**Поля ввода:**
- длина стержня L (м)
- температура левого конца TL (°C)
- температура правого конца TR (°C)
- количество узлов N

**Результаты:**
- график температуры вдоль длины стержня
- визуализация стержня с цветовой индикацией (синий - холодно, красный - горячо)
- максимальная и минимальная температура
- температура в выбранной точке

**Алгоритм решения:**
- Разбиение стержня на N-1 равных участков с шагом Δx = L/(N-1)
- Расчёт температуры в каждом узле по формуле: T_i = TL + (TR-TL)/L * x_i, где x_i = i * Δx
- Построение графика и цветовой визуализации

### Страница «Движение парусной лодки» (boat.tpl)

**Поля ввода:**
- скорость течения реки vcurrent (м/с)
- скорость ветра vwind (м/с)
- коэффициент парусности kwind (Н·с²/м²)
- коэффициент сопротивления воды c (Н·с²/м²)

**Результаты:**
- оптимальный угол паруса θopt (град)
- максимальная скорость лодки vmax (м/с)
- график зависимости скорости лодки от угла паруса v(θ)

**Алгоритм решения:**
- Ввод исходных данных от пользователя
- Определение функции расчёта скорости лодки для заданного θ:
  - Начальное приближение скорости лодки vboat = (vcurrent, 0)
  - Вычисление относительной скорости воды vrel,water = vboat - (vcurrent, 0)
  - Сила сопротивления воды Fdrag = -c * |vrel,water| * vrel,water
  - Относительная скорость ветра vwind,rel = vwind - vboat
  - Угол относительного ветра β = arctan2(vwind,rel,y, vwind,rel,x)
  - Угол паруса θ в радианах, угол атаки φ = θ - β
  - Сила ветра: Fwind = kwind * u^2 * max(0, sinφ), где u = |vwind,rel|
  - Направление силы ветра перпендикулярно парусу (угол θ+90°)
  - Решение уравнения равновесия Fwind + Fdrag = 0 методом простой итерации
- Оптимизация угла паруса:
  - Перебор углов θ от 0° до 180° с шагом 1°
  - Для каждого θ вычисление установившейся скорости лодки
  - Нахождение максимальной скорости и соответствующего угла

## Установка и запуск

### Требования к окружению

- Python 3.8 или выше
- pip (менеджер пакетов Python)
- Git (для клонирования репозитория)
- Браузер (Chrome, Firefox, Edge — последние версии)

### Необходимые библиотеки

```bash
pip install bottle matplotlib numpy
```

### Инструкция по установке и запуску

1. Клонирование репозитория
   
```bash
git clone https://github.com/sqtwix/BottleWebProject_C324_2_PSSH.git
cd BottleWebProject_C324_2_PSS
```

2. Запуск приложения

```bash
python main.py
```

3. Открытие в браузере

Перейдите по адресу: http://localhost:5555
